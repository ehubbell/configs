## Overview
A Prettier config for various projects.

## Installation
```
npm i @ehubbell/prettier-config
npx install-peerdeps --dev @ehubbell/prettier-config
```

## Usage
Create an `.eslintrc` file and add the following:

```json
{
  "root": true,
  "extends": ["@ehubbell/prettier-config"]
}
```
