## Overview
An eslint config for Web projects.

## Installation
```
npm i @ehubbell/eslint-config-web
npx install-peerdeps --dev @ehubbell/eslint-config-web
```

## Usage
Create an `.eslintrc` file and add the following:

```json
{
  "root": true,
  "extends": ["@ehubbell/eslint-config-web"]
}
```
