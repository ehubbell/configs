## Overview
An eslint config for Web projects.

## Installation
```
npm i @ehubbell/eslint-config-web --save-dev
npm install
```

## Usage
Create an `.eslintrc` file and add the following:

```json
{
  "root": true,
  "extends": ["@ehubbell/eslint-config-web"]
}
```

Or, add the following to your `package.json` file:
```json
  "eslintConfig": {
    "extends": "@ehubbell/eslint-config-web"
  },
```
